Arrays: Convenient lists that will make your code easier to write

Before you begin:

Make sure you're very comfortable with loops, especially for loops. If you can successfully type a for loop right now without looking at any examples, you're ready to go. If not, you'll want to practice some more.

Ready? Let's do this.

What is an array?

An array is a list of things.

Those things can be strings, numbers, variables, functions, or even other arrays.

An array is formatted as a comma-separated list of things inside square brackets, and it looks like this:

['thing 1', 'thing 2', 'thing 3']

Basic array: Conveniently store a list of names

Let's say you have a list of names. Instead of creating a variable for each one and then having to keep track of all of those variables, you could make one array and store the whole list.

Type the following example to see how it works:


			var names = ['Bowser','Ganon','Shmebulock'];
			console.log(names[0]);

You'll notice that names[0] will give you 'Bowser'. Try replacing the 0 with a 1 or a 2. What happens?

Array index

When you see something like names[2], the number is called the index, and it gives you the position of an item in the array.

Warning about the array index:

The array index starts counting at 0. Many errors happen because people forget that the first item in the array is 0.

Array length

Let's say you want to get the length of your list. Here's how that would work (type it in):


			var names = ['Bowser','Ganon','Shmebulock'];
			console.log(names.length);

Change a value in an array

To change a value in an array, all you need to do is set it to something else like this (type it in):


			var names = ['Bowser','Ganon','Shmebulock'];
			names[1] = 'Link';
			console.log(names[1]);

Examples of stuff you can put in an array

You can put strings, numbers, arrays, variables, and other stuff inside an array. Type the following example to try it out:


			var x = 'this is x';
			var stuff = ['strings work', 12, x, ['inner array item 1', 'inner array item 2'], 'last item'];

			console.log( stuff[0] ); // string
			console.log( stuff[1] ); // number
			console.log( stuff[2] ); // variable
			console.log( stuff[3] ); // array
			console.log( stuff[3][0] ); // string inside array
			console.log( stuff[3][1] ); // other string inside array
			console.log( stuff[4] ); // string

If you want to a function inside an array, you can do that too! Try the following example:


			var stuff = ['first item', function(){ alert('hi'); }, 'last item'];

			console.log( stuff[0] ); // string
			console.log( stuff[1] ); // function

			stuff[1](); // actually run the function

How to loop through an array

When you want to do something with each value of an array, but you don't want to write the code over and over, you can use a loop.

Try the following example to see how it works:


			var names = ['Bowser','Ganon','Shmebulock'];

			for(var i=0; i<names.length; i++){
				console.log('Hello there' + names[i]);
			}

This code will go through each name and show a greeting to that name inside the console.

Exercises:

  1. Type the following array 3 times. Once you're comfortable, type it again 5 times without looking at an example.
    2. var items = ['a','b','c'];
  2. Create an array of any three items and display the first one in a div. Try displaying the second one and the third.
  3. Create an array of two items and display the array's length in a span.
  4. Create an array, change the value of one of the items, and show the new value on screen (can be a div or a span or any HTML element).
  5. Create an array of at least three names and use a loop to show a greeting to each one in a div. Repeat this exercise until you can loop through an array without looking at an example.
